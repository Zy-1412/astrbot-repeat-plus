# RepeatPlus-Enterprise — QQ群复读增强插件

AstrBot QQ 群复读增强插件，支持文本/图片/表情复读、模糊匹配、趣味打断、排行榜，以及**抽老公/老婆双模式 + 关系图 + 求婚系统**，提供 9 组 43 项可视化后台配置。

## 功能速览

- **智能复读** — 文本/图片/表情复读，模糊匹配，权重衰减，权重扰乱
- **趣味打断** — 可配置的打断策略，支持自定义打断词库和概率
- **排行榜** — 日榜/周榜/月榜/总榜，多维度统计
- **抽老公/老婆** — 双模式独立配置，每日抽取 + 强娶 + 求婚 + 排行榜
- **羁绊关系图** — 力导向布局可视化，并发头像下载，支持 2~50+ 人零重叠渲染
- **数据持久化** — 排行榜/抽取记录/活跃数据自动保存，跨重启不丢失

## 配置面板

插件提供 **9 组 43 项**可视化配置，在 AstrBot WebUI 管理面板中直接调整：

| 配置组 | 项数 | 主要内容 |
|--------|------|----------|
| 🎯 复读判定 | 7 | 触发阈值、窗口大小、模糊匹配灵敏度、权重衰减、最小消息长度 |
| ⏱️ 冷却与抑制 | 4 | 冷却时间、重复抑制、冷却递增、快速触发衰减 |
| 🎭 打断系统 | 6 | 打断概率、打断词库、打断策略、延迟范围 |
| 🎨 复读风格 | 3 | 复读前缀/后缀、风格模式 |
| 🛡️ 过滤 | 3 | 白名单/黑名单群组、忽略用户 |
| 🔧 系统 | 2 | 调试模式、数据保存间隔 |
| 💕 抽取设置 | 9 | 每日上限、抽取范围、活跃天数、记录上限、名字替换 |
| 💍 强娶与求婚 | 5 | 强娶 CD、强娶次数、求婚开关、接受/拒绝话术 |
| ⚙️ 高级选项 | 6 | 人类延迟、排行榜保留天数、长度偏差、最大窗口 |

## 安装

1. 下载 `astrbot_plugin_repeat_plus_v1.2.2.zip`
2. 在 AstrBot WebUI → 插件管理 → 上传插件包
3. 启用插件，在管理面板中配置参数
4. 在群内发送 `/复读开启` 启动复读功能

## 依赖

- AstrBot `>=4.0.0`
- Pillow（关系图生成，可选）— `pip install Pillow`
